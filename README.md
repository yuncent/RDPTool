# RDPTool (RDP端口修改工具)

这是一个轻量级、绿色的 Windows 远程桌面 (RDP) 端口修改工具，专为提升服务器安全性和简化运维操作设计。

### 项目特点
- 全系统兼容：支持从 Windows XP 到 Windows 11 的所有主流版本。
- 呼吸感布局：基于 320px 黄金宽度设计，采用 25px 宽边距与控件间距，视觉规整。
- 智能随机化：内置一键生成 10000-60000 之间高位安全端口的逻辑。
- 权限自提升：程序启动时自动请求管理员权限，确保注册表修改成功。
- 交互式反馈：集成状态栏实时显示操作进度，并提供便捷的系统远程设置跳转。

### 使用方法
1. 下载并以管理员权限运行 `RDPTool.exe`。
2. 在输入框内输入目标端口，或点击“随机端口”自动生成。
3. 点击“确认修改”按钮。
4. 弹出成功提示后，重启计算机以使新端口生效。
5. 注意：请务必确保您的防火墙已手动放行新的端口号。

### 编译说明
本项目采用原生 C++ 与 Win32 API 编写，无任何第三方库依赖。

建议编译命令 (MinGW)：
g++ RDPTool.cpp -o RDPTool.exe -O2 -mwindows -static -ladvapi32 -lshell32
